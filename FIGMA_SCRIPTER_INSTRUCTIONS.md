# Инструкция по использованию скрипта для Figma Scripter

## Подготовка

1. Установите плагин **Scripter** в Figma (если еще не установлен)
2. Откройте файл `figma-scripter-activity-dashboard.js`

## Использование

1. **Создайте фрейм:**
   - Размер: `393x852` (или больше)
   - Название: любое (например, "Activity Dashboard")

2. **Выделите фрейм** (кликните на него)

3. **Запустите Scripter:**
   - Плагины → Scripter → Run Script
   - Вставьте содержимое файла `figma-scripter-activity-dashboard.js`
   - Нажмите Run

4. **Результат:**
   - Скрипт создаст полный Activity Dashboard с:
     - Header "Today" с кнопками назад и редактирования
     - Заголовок "Activity"
     - Карточка "Cardio Load" с значением и иконкой сердца
     - Карточка "Exercise Days" с календарем и деталями тренировки
     - Карточка "Active Zone Min" с прогресс-баром

## Токены

Скрипт использует следующие токены (можно изменить в начале файла):

```javascript
const TOKENS = {
  colors: {
    bg: "#F6F7F2",           // Фон экрана
    card: "#FFFFFF",         // Фон карточек
    accent: "#007A7A",       // Акцентный цвет
    accentMuted: "#D6EBEB",  // Приглушенный акцент
    textPrimary: "#1A1C1E",  // Основной текст
    textSecondary: "#6C727A" // Вторичный текст
  },
  radius: 24,               // Радиус скругления
  padding: 20               // Внутренние отступы
};
```

## Особенности

- ✅ Все элементы используют Auto Layout где возможно
- ✅ Правильные размеры шрифтов (48px для больших чисел, 16px для заголовков)
- ✅ Корректные цвета из токенов
- ✅ Календарь с прямоугольниками 24x48px и radius 12px
- ✅ Круговой прогресс-бар для Active Zone Min

## Устранение неполадок

**Ошибка "Выдели фрейм":**
- Убедитесь, что вы выделили фрейм перед запуском скрипта

**Ошибка загрузки шрифта:**
- Убедитесь, что шрифт Inter установлен в системе
- Или измените `family: "Inter"` на доступный шрифт

**Элементы не на месте:**
- Убедитесь, что фрейм имеет размер минимум 393x852
- Проверьте, что Auto Layout включен для карточек
