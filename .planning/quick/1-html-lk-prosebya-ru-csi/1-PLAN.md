# Quick Task 001: Создать HTML-копию lk.prosebya.ru

**Goal:** Создать статичный HTML-файл `prosebya-prototype-base.html`, визуально копирующий главную страницу lk.prosebya.ru, как основу для прototипа CSI-опросника.

## Tasks

### Task 1: Создать HTML-файл
- **File:** `prosebya-prototype-base.html`
- **Action:** Воспроизвести структуру и дизайн страницы на основе скриншотов и accessibility snapshot
- **Content:**
  - Header с логотипом просебя и гамбургер-кнопкой
  - Hero banner (синий фон, "Станьте опорой для себя", кнопка "Подобрать трек")
  - Секция "Все материалы для самопомощи" с фильтрами-тэгами
  - 5 контент-секций с карточками (тесты, медитации, практики, видео, статьи)
  - Футер с реквизитами
- **Tech:** Tailwind CSS CDN + Google Fonts (Inter Cyrillic)
- **Done:** Файл открывается в браузере и визуально соответствует оригиналу
