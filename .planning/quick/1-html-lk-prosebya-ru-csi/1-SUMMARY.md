## What was done

Создан файл `prosebya-prototype-base.html` — статичная HTML-копия главной страницы lk.prosebya.ru.

### Реализованные элементы:
- **Header**: логотип «просебя» (желтый), кнопка гамбургера
- **Hero banner**: синяя карточка с сеткой, заголовок «Станьте опорой для себя», бейдж «40+ треков», желтая кнопка «Подобрать трек»
- **Фильтры**: таблетки категорий (Все активен, остальные неактивны)
- **Секция «Психологические тесты»**: 5 горизонтально скроллируемых карточек
- **Секция «Медитации»**: 4 градиентные карточки
- **Секция «Практики самопомощи»**: 7 карточек
- **Секция «Меня всё раздражает и бесит»**: 3 видео-карточки с аватаром + плей-кнопкой
- **Секция «Статьи для саморазвития»**: 4 статьи-карточки с тегом «Статья»
- **Футер**: реквизиты ООО «ПРОСЕБЯ», ссылки

### Tech stack:
- Tailwind CSS v3 (CDN)
- Google Fonts — Inter (кириллица)
- Чистый HTML без зависимостей
